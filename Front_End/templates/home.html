This project presents a Grad-CAM Assisted Eye Disease Classification System using deep learning techniques. The system employs two independent CNN models trained on OCT and fundus image datasets.

The objective is to assist ophthalmologists by providing accurate, automated retinal disease diagnosis with visual interpretability.

  • Model 1: OCT C8 – 8-class classification with Grad-CAM
  • Model 2: EyeDisease101 – 6-class fundus classification
