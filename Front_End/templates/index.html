🏠 Home

This project implements an intelligent eye disease detection system using deep learning techniques. The system supports two different retinal imaging datasets and applies dedicated models for disease prediction.

Model explainability is provided using Grad-CAM for selected models to highlight important regions influencing predictions.

⚠ This system is intended for academic and research purposes only.

📊 Datasets

  • Retinal OCT C8 – OCT images used for multi-class retinal disease classification (AMD, CNV, DME, DR, etc.).
  • Dataset-101 – Fundus images used for eye disease detection including Cataract and Glaucoma.

📘 Instructions

  1. Navigate to the required model section.
  2. Upload a retinal image (OCT or Fundus).
  3. Click the prediction button.
  4. View prediction results.
  5. Grad-CAM visualization is shown only for Model 1.
